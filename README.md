# SOC Analyst Casebook

Welcome to my SOC Analyst Casebook.

This repository documents my journey as I train to become a security analyst. Rather than simply completing courses, I want to build a collection of real investigations, notes, and projects that reflect how my skills and thinking develop over time.

## Purpose

I created this repository for two reasons.

First, for myself.

Growth has always been important to me, and I want a record of my progress. I want to look back at my earliest investigations, compare them with future ones, and see how much I've improved.

Second, for anyone interested in my work.

Whether you're a recruiter, mentor, fellow learner, or cybersecurity professional, I hope this repository demonstrates my commitment to continuous learning, disciplined practice, and evidence-based investigation.

## What You'll Find Here

As this repository grows, it will include:

- Phishing investigations
- Incident response case studies
- Security methodologies and notes
- Documentation of tools used during investigations
- Lessons learned from each case

Every investigation is documented to show not only my conclusions, but also the reasoning and evidence behind them.

## My Goal

Six months from now, I want this repository to reflect consistency, dedication, curiosity, and completed work.

This is more than a collection of files.

It's a record of my growth as a future cybersecurity professional.
